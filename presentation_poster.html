Visual Telemetry & Analysis

AI Interviewer tracks candidate expressions, gaze, and vocal metrics during live interview rounds. The visual engine generates real-time telemetry overlays and emotion vector distributions.

Visual Attention Tracking

Calculates live gaze orientation and eye-contact ratios.

Facial Emotion Profiling

Tracks micro-expressions (happy, neutral, surprise, sad, etc.).

Dynamic Textarea Fallback

Provides manual input overrides if mic/camera capture is restricted.

AI Processing Pipeline

1

Webcam/Mic Authorization: Request browser media permissions for candidate capture.

2

Real-time Telemetry Processing: Stream video frames to visual tracking adapters.

3

Vocal & Keyword Grading: Transcribe speech, measure WPM, and match semantic overlap.

03

Reflection & Scope

Key Advantages

Eliminates reviewer bias, providing standardized objective scoring. Features a resilient cloud backend architecture with lightweight SQLite database local fallbacks.

System Limitations

Relies on client-side text extraction when hosting environments restrict API requests. Requires a stable video stream for eye-gaze tracking metrics.

Future Scope

Integrating local Hugging Face LLM models for deeper situational assessments. Adding candidate voice sentiment analysis using small acoustic classifiers.

System Integration

Built with modular Flask Blueprint routes, enabling seamless plug-and-play onboarding of institutional student databases and management dashboards.
